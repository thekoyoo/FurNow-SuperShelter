--------------------------------------------------------------------------------
Power Source
---------------------------------------


Uses the same one as SolarCat. Only the battery is required.

- Charge controller (10A is well over enough for this build) 
- Solar panel (10W)
- 12V Battery (Ah rating corresponds to battery life, I used 6Ah)


Other electronic components
---------------------------------------
                 
- AM312 PIR Motion Sensor (can also get the HC-SR505)
- ESP32-S3 Sense (any esp32 with a camera + supports psram will do but will run slower)
- 25v 100uf electrolytic capacitors
- Stepper motor
- 3.3v buck converter
- tmc2209 motor driver (better than 2208) (a4988 is a cheaper but worse alternative; it’s pretty much plug and play the same except for setting up the VREF)

Misc
--------------------------------------------------------------------------------

- Drill/Razor blade (no specific recommendation here)
- Soldering and crimping tools (crimping is optional but recommended for 12V battery connections if you dont have the proper wires)
- 3d Printer/wood/plank/sheet of material that isn’t flat
- String (nylon/fishing line preferred)
- Data transfer usb cable + computer
